## 4. 如何正确学习刷题
* 解体步骤
    * 5分钟读懂及与面试官确认要解决的问题
    * 列出所有可能的解法并思考时间空间复杂度
    * 选出最优解法并编写解代码
    * 检查代码逻辑与再次确认边界
    * 提交给面试官
* 五毒神掌（同一道题要分阶段刷5次）
    * 自我完成或看题解后
    * 默写
    * 一天后
    * 一周后
    * 面试前
